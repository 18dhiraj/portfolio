# **App Name**: Craft Gallery

## Core Features:

- Project Gallery: Display creative projects in a visually appealing grid.
- Detailed Project View: Showcase project images/media and project descriptions.
- About Me Section: Introduce yourself with a personal bio and relevant details.
- Contact Form: Allow visitors to send you messages and inquiries.

## Style Guidelines:

- Primary color: Teal (#008080) for highlights and interactive elements.
- Background color: Off-white (#F0F0F0) for a clean, modern look.
- Accent color: Light gray (#D3D3D3) for subtle contrast and visual interest.
- Body and headline font: 'Inter' sans-serif for a modern and neutral feel.
- Use simple, line-based icons for navigation and visual cues.
- Employ a grid-based layout for organizing project images and content.
- Use subtle transitions and animations to enhance the user experience.
